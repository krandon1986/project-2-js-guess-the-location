<h1 align="center">Guess the Location</h1>

## Description 

This game is loosely based on the combination of the games, 'Guess the Word' and 'Hang Man'. But instead of words, the player has to guess the location. The location can be a city, a country or a popular place. The player are given a hint on what the location is and they have to fill in the empty field by clicking the letter buttons to input it to the board. The correct letter is put in the board and the selected letter button will colored green, but incorrect letter result in a life being taken out and the letter button being colored red.
